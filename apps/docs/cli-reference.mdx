## tpl watch

Watch for `.tpl.*` changes and regenerate automatically (~50 ms debounce).

```bash
tpl watch
# or
tpl w
```

The primary workflow for development. Generates on startup, then re-generates within ~50 ms of any `.tpl.*` save.

Options:

| Option | Description |
|--------|-------------|
| `--cwd <dir>` | Working directory (defaults to `process.cwd()`) |
| `--output <path>` | Override the manifest file path |

Recommended for dev scripts:

```json
{
  "scripts": {
    "dev": "tpl watch & next dev"
  }
}
```

## tpl generate

Generate sibling `*.tpl.gen.ts` files and the `lib/tpl.gen.ts` manifest from all `.tpl.*` files (one-shot).

```bash
tpl generate
# or
tpl g
```

Options:

| Option | Description |
|--------|-------------|
| `--cwd <dir>` | Working directory (defaults to `process.cwd()`) |
| `--output <path>` | Override the manifest file path |

Use in CI / build scripts before your TypeScript compiler:

```json
{
  "scripts": {
    "build": "tpl generate && next build"
  }
}
```
